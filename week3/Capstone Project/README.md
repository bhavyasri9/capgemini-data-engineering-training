# Capstone Project – Cloud Data Analytics Platform

## Overview

This is a team-based capstone project developed by a group of 7 members. The project focuses on building a centralized data analytics platform using Microsoft Azure services.

---

## Problem Statement

Organizations struggle with:

* Data silos
* Lack of real-time insights
* Inefficient reporting

This project aims to solve these issues by creating a unified data pipeline and analytics system.

---

## 👥 Team Details

* Team Size: 7 Members
* Collaborative development with task distribution

---

## Tech Stack

* Microsoft Azure
* Azure Databricks
* Azure Data Factory
* Azure Data Lake Storage
* Power BI

---

## Architecture

Data Sources → Data Factory → Databricks → Data Lake → Power BI Dashboard

---

## Workflow

1. Data ingestion from multiple sources
2. Data transformation using Databricks
3. Data storage in Data Lake
4. Dashboard creation using Power BI

---

## Key Features

* Centralized data processing
* Scalable architecture
* Interactive dashboards
* Real-time insights

---
