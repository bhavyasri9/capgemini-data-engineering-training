# Azure Databricks Data Engineering Project

## Overview

This project demonstrates an end-to-end data engineering pipeline built using Microsoft Azure and Azure Databricks. The goal is to transform raw data into clean, structured, and analytics-ready datasets.

---

## Objective

* Build a scalable ETL pipeline
* Implement Medallion Architecture (Bronze–Silver–Gold)
* Generate business-ready insights

---

## Tech Stack

* Microsoft Azure
* Azure Databricks
* Azure Data Factory
* Azure Data Lake Storage
* Python, PySpark

---

## Learnings

* Cloud-based data engineering
* Distributed data processing
* ETL pipeline design

---

## Challenges

* Building a connection between Azure file system storage and system files using integration runtime
* Debugging Spark jobs
* Integrating Azure services

---

## Conclusion

This project provided hands-on experience in building scalable data pipelines using modern cloud technologies.
