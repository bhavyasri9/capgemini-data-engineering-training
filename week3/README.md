# Week 3 – Data Engineering using Azure Databricks

## Overview

This week focused on building cloud-based data engineering pipelines using Microsoft Azure and Azure Databricks.

The work includes:

* Individual Project (End-to-End Pipeline)
* Team Capstone Project (7 Members)

## Individual Project – Azure Databricks Implementation

### Objective

To design and implement a scalable ETL pipeline that transforms raw data into structured, analytics-ready datasets.

### Tech Stack

* Azure Databricks
* Azure Data Factory
* Azure Data Lake Storage
* Python
* PySpark

### Architecture (Medallion Architecture)

Raw Data → Bronze Layer → Silver Layer → Gold Layer → Analytics

### Workflow

1. Data ingestion into Data Lake (Bronze Layer)
2. Data cleaning & transformation (Silver Layer)
3. Data aggregation & business logic (Gold Layer)

### Key Features

* Implemented ETL pipelines using PySpark
* Performed data cleaning (handling nulls, removing duplicates)
* Applied schema standardization
* Optimized transformations for performance

### Outcome

* Generated high-quality structured datasets
* Enabled business-ready insights for analysis
