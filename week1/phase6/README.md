# Phase 6

## Objective

The objective of this phase was to strengthen PySpark skills by working on joins, window functions, date operations, and building a complete data pipeline.

## What I Implemented

* Cleaned and validated datasets by handling null values and invalid records
* Performed different types of joins, including inner, left, and anti joins
* Applied aggregations to calculate total spend and order counts
* Used window functions like rank, running total, and lag
* Performed date-based analysis, such as monthly sales aggregation
* Built a complete ETL pipeline from data loading to saving output

## Key Learnings

* Improved understanding of joins and data validation
* Gained hands-on experience with window functions
* Learned how to debug data issues effectively
* Built confidence in creating end-to-end pipelines

## Challenges Faced

* Handling invalid foreign keys
* Understanding window function behavior
* Managing transformations step-by-step

## Conclusion

This phase helped me achieve fluency in PySpark and prepared me for working on real-world data engineering projects in Databricks.
