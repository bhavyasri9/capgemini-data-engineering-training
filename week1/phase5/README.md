# Phase 5 – Databricks Olist Data Pipeline

## Objective

The objective of this phase was to work with a real-world dataset and build an end-to-end data pipeline using Databricks and PySpark. This phase focuses on handling multiple datasets, performing complex transformations, and generating meaningful business insights.

## What I Implemented

* Loaded multiple datasets from Databricks volume
* Performed data validation using schema checks and sample records
* Cleaned data by handling null values and casting columns to appropriate data types
* Joined multiple datasets to create a unified dataset for analysis

### Business Transformations

* Calculated **Top 3 customers per city** using window functions
* Computed **running total of sales** over time
* Identified **top products per category**
* Calculated **Customer Lifetime Value (CLV)**
* Performed **customer segmentation** based on total spending (Gold, Silver, Bronze)
* Built a **final reporting dataset** combining customer, order, and spending insights

## Key Learnings

* Learned how to work with real-world multi-table datasets
* Understood the importance of data relationships and joins
* Gained hands-on experience with **window functions** and advanced aggregation

## Challenges Faced

* Understanding relationships between multiple datasets
* Handling large data joins efficiently

## Conclusion

This phase helped me transition from working on small datasets to handling real-world data engineering problems.
