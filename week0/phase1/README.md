#Phase1 – SQL to PySpark
To learn basic SQL queries and convert them into PySpark.

## I have learned the exercises
- Select queries
- Filter conditions
- Column selection
- GroupBy operations
  
main concepts are:
- PySpark DataFrame works like SQL tables
- filter() is equivalent to WHERE
- select() is equivalent to SELECT
- groupBy() is equivalent to GROUP BY

## Challenges
- Understanding PySpark syntax
- Converting SQL logic into DataFrame operations
